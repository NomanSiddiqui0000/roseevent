# Rose Events

Premium Event Management & Luxury Wedding Planning platform.

## Features

- **Luxury Weddings**: Bespoke planning and timeless designs for your special day.
- **Corporate Events**: Seamless execution, elegant branding, and professional environments.
- **Private Parties & Birthdays**: Exquisite details, customized themes, and curated experiences.
- **Process Walkthrough**: Clear, step-by-step methodology outlining consultation, design, and coordination.
- **Portfolio & Testimonials**: Visual gallery of past premium celebrations and client feedback.

## Tech Stack

- **Frontend Framework**: React with TypeScript
- **Styling**: Tailwind CSS for modern layouts
- **Build Tool**: Vite
- **Icons**: Lucide React
